# Warehouse项目

这是一个完整的Android项目，项目中涉及到公司信息及个人信息已删除。项目是基于PDA，对商品上下架、转仓、报损进行处理等操作。项目涉及到的技术点包括：

* RecyclerView的使用，包括：item包含Spinner、EditText，为RecyclerView添加点击删除事件
* 使用OkHttp作为网络请求框架
* 使用ButterKnife作为注解框架
